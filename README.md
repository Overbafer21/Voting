# 🗳️ Симулятор электронного голосования (Java Swing)

Небольшой учебный проект на **Java Swing**, который имитирует работу электронной урны для голосования.

## Что умеет приложение

- Голосование за трёх кандидатов (A, B, C).
- Мгновенное обновление счётчиков после каждого нажатия.
- Отображение итогового количества голосов.
- Кастомный тёмный интерфейс (не стандартный шаблон): карточки кандидатов, акцентные кнопки и блок статистики.

## Дизайн

В этой версии дизайн изменён относительно базового примера:

- тёмная цветовая тема;
- карточный вид для кандидатов;
- современная типографика и отступы;
- отдельный блок итоговой статистики (включая общее количество голосов).

## Как запустить

1. Убедитесь, что установлен JDK 11+.
2. Скомпилируйте проект:

```bash
javac VotingMachineSimulator.java
```

3. Запустите:

```bash
java VotingMachineSimulator
```
